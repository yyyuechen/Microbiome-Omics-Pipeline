# chimney-microbiome-omics-pipeline
A collection of bioinformatics scripts and pipelines used for the analysis of metagenomic and metatranscriptomic datasets from active chimney system in the Red Sea, supporting genome reconstruction (MAGs) and functional characterization of microbial communities.

# project overview
This repository contains scripts and workflows used for the analysis of metagenomic and metatranscriptomic datasets from active chimney samples in the Red Sea. The pipeline supports genome-resolved analyses, including metagenome assembly, binning, taxonomic classification, and functional profiling.
